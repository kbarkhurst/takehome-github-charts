Github Dashboard

This dashboard visualizes the Forks, Watchers and Stars for popular frameworks on GitHub. Using vanilla javascript, this project utilizes Axios and Chart.js to access the GitHub API and display these javascript repo's engagement figures.

Make the numbers change. Head on over to a repo and add a Fork, Watch or Star: Vue, Angular, Ember, Svelte, React.
