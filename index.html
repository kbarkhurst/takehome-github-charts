Github API Dashboard

This dashboard provides data visualization of Github Forks, Watchers, and Stars for popular javascript frameworks.

Using vanilla javascript, this project utilizes Axios and Chart.js to access GitHub's API and to display the provided engagement figures for each included javascript framework in the charts below.

Make the numbers change.

Head on over to a repo and add a Fork, Watch or Star, then check the numbers below: Vue, Angular, Ember, Svelte, React.
